# Demenagement-paris
Paris Express Déménagement est née en 2012 d'une rencontre entre Julien Marchand et Éric Beaumont, deux amis aux parcours complémentaires — l'un logisticien, l'autre entrepreneur. Frustrés par la mauvaise réputation du secteur du déménagement, ils ont décidé de prouver que ce métier pouvait être exercé avec excellence.
